GTSReader est un logiciel permettant de visualiser des modèles 3D au format gts (GTS : GNU Triangulated Surface)
et d'utiliser ces modèles de manière intuitive.

SOURIS
Bouton gauche : faire pivoter le modèle
Bouton droit : déplacer le modèle
Molette haut : zoom +
Molette bas : zoom -

CLAVIER
z : Rotation haut
s : Rotation bas
q : Rotation gauche
d : Rotation droite
o : Translation haut
l : Translation bas
k : Translation gauche
m : Translation droite
F1 : Affichage du modèle en points
F2 : Affichage du modèle en segments
F3 : Affichage du modèle en faces
- : Zoom -
+ : Zoom +

GESTION DES MODELES
Il y a 2 façons de charger les modèles GTS dans le logiciel : 

Methode 1
Une liste de modèles est disponible au lancement du logiciel. Si vous souhaitez en rajouter
ou en enlever, le dossier à modifier est ressources/gts

Methode 2
Vous pouvez importer vos propres modèles, soit par drag'n drop, soit via Fichier > Ouvrir

BARRE DE RECHERCHE ET DE FILTRAGE PAR TAG
Une barre de recherche vous permettra de filtrer les modèles par nom.

Il est possible, pour chaque modèle, d'ajouter/supprimer 
un ou plusieurs tags via le sous-menu Outils/Modeles/...
Pour retrouver les modèles associés un à un tag, il vous suffira
de rentrer votre tag dans la barre de recherche en le précédant par le caractère #

Exemple : #pokemon

Pour voir l'intégralité des tags sur chaque modèle: Outils > Modeles > Lister les tags

EXPORT
Il est possible de prendre des screenshots du modèle courant en allant dans Fichier > Exporter.

AFFICHAGE
Le logiciel inclut une gestion de la lumière et des ombrages sur les faces plus ou moins éloignées
de la source de lumière, pour un rendu très réaliste.

Le mode plein écran est disponible dans la rubrique Affichage du menu.

Vous pouvez changer l'image de fond via Affichage > Changer image de fond. 
Des images sont disponibles dans ressources/images, mais vous 
pouvez également selectionner vos propres images.

Vous pouvez également changer la couleur des modèles grâce aux différentes rubriques du menu Outils : 
 - couleur aléatoire uniforme
 - couleur aléatoire de votre choix
 - couleur aléatoire par face
 - dégradé de couleurs
